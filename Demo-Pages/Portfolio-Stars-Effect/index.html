Brandon Helm Introduction

I'm a passionate web developer with a focus on creating engaging and dynamic user experiences.

I have specific interest in the ways WebGL can be used to create immersive graphics and interactive elements on the web.

While I am capable of working with numerous technologies to facilitate polished front end UI/UX, I am a full-stack developer.

I'm familiar with Restful APIs, database management, and server-side programming to build robust web applications.

My featured project is called Strategic Pivot. It has some lofty goals to facilitate early stage project networking through currating learning content with interactive quizzes. It then diagrams out relationships between aspects of a pitch or approach. I believe this will greatly aide in national project managemement, and is a corner-stone of government efficiency.
