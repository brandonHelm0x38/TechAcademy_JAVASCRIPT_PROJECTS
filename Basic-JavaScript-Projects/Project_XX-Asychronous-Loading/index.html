Basic JavaScript Projects - Project 10: Asynchronous Loading

Hello World

This project demonstrates asynchronous loading of JavaScript files. The script is loaded in parallel with the HTML parsing, allowing for faster page load times and improved performance.

Asynchronous loading is particularly useful for large scripts or when multiple scripts are being loaded, as it prevents blocking the rendering of the page.

In this example, the script "async.js" is loaded asynchronously, meaning it does not block the HTML parsing and will execute as soon as it is downloaded.

The scripts "defer1.js" and "defer2.js" are loaded with the "defer" attribute, meaning they will only execute after the HTML parsing is complete. This ensures that the DOM is fully constructed before the scripts run.

Deferred scripts will be loaded and executed in the order they appear in the document.

Asynchronous and deferred loading can significantly improve the user experience by reducing the time it takes for the page to become interactive.

Note the document.write() method will not work when scripts are loaded asynchronously or deferred.
